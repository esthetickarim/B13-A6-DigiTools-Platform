DigiTools – Digital Tools Buying Platform

Project Description
DigiTools is a modern web application where users can explore and purchase premium digital tools. The platform is designed to improve productivity and creativity by providing a curated collection of tools with a smooth user experience.

Users can browse products, add them to the cart, view total pricing, and complete checkout with interactive notifications.

Technologies Used
React.js
Tailwind CSS & DaisyUI
JavaScript (ES6+)
React-Toastify (for alerts & notifications)
JSON (for product data)
Vite (for fast development)

Key Features

1. Add to Cart Functionality
Users can add products to the cart with instant toast notifications. Duplicate items are prevented with warning alerts.


2. Remove & Checkout System
Users can remove individual products from the cart or clear all items using the "Proceed to Checkout" button.
